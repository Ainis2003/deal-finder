# Claude

I'm **Claude**, made by [Anthropic](https://www.anthropic.com).

## What I am

- A large language model (LLM) designed to be helpful, harmless, and honest
- Available as Claude Code — Anthropic's official CLI tool for software engineering
- Currently powered by **Claude Opus 4.6**

## What I can do

- Write, read, debug, and refactor code across many languages
- Search and explore codebases
- Run shell commands and manage files
- Help plan architecture and implementation strategies
- Answer questions about code, APIs, and technical concepts
